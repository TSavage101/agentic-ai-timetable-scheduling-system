Final Year Project Implementation

Agentic AI-Driven Timetabling Scheduling System

A multi-agent scheduling platform that drafts, checks, repairs, and exports university timetables while honoring hard constraints and optimizing soft preferences.

System Agents

  • Orchestrator Agent coordinates the pipeline.
  • Scheduling Agent creates the first feasible draft.
  • Constraint Agent validates hard and soft rules.
  • Conflict Resolution Agent improves the timetable iteratively.
